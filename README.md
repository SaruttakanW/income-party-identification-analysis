# Annual Household Income and Political Party Identification Analysis

## Overview
This project investigates whether annual household income is associated with political party identification among five ethnic minority voters in the UK.

## Dataset
British Election Study Ethnic Minority Survey, 2010

Due to data usage restrictions, the dataset cannot be publicly redistributed.

## Tools
- R
- dplyr
- haven
- rcompanion

## Methods
- Chi-square Test
- Cramér’s V
